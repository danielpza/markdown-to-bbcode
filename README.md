# markdown-to-bbcode

Convert from markdown to bb code (text formatting used in steam)

## Usage

```sh
markdown-to-bbcode < README.md > out.txt
```

## Installing

Available from github

```sh
npm i danielpza/markdown-to-bbcode
```
